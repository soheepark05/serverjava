JSP(Java Server Page)

Html 형식에서 자바 코드를 사용할 수 있는 "자바 언어"로
서블릿에서 java 코드로 응답 HTml을 구현했던 복잡함을 간단하게 작성, 해결할 수 있다.

1. jsp 스크립트 요소(Elements)

jsp페이지에 자바 코드를 직접 기술할 수 있게 해주는 기능이다.

  1. 선언문 (<%! 자바 코드 %>): 멤버 변수와 메소드를 선언하기 위해서 사용한다.
  2. 스크립트릿(<% 자바 코드%>): JSP에서 자바코드를 사용하기 위해서 사용한다.
  3. 표현식(<%=자바 코드 %>): 자바에서 작성한 값을 출력하기 위해서 사용한다.

2.지시어(Directive)

JSP 페이지 전체에 영향을 미치는 정보를 기술 할 때 사용한다.

  1. page: 현재 JSP페이지를 컨테이너에서 처리하는데 필요한 각종 속성을 기술한다. 보통 맨앞에 위치함
    • language: jsp페이지내부에서 사용할 스크립트 언어를 지정한다,
    • contentType: 웹브라우저가 받아 볼 페이지의 형식, 인코딩 방식을 지정한다.
    • pageEncoding
    • import
    • errorPage
    • isErrorPage
    • 등등...
  2. include: jsp페이지에서 다른 jsp 파일 또는 html 파일을 삽입할 수 있도록 하는 지시어 이다.
  3. taglib: jsp기능을 확장할 수 있는 사용자 정의 태그를 가져와서 사용할 수 있도록 하는 지시어이다.
